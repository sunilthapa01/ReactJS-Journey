# React Practice Tasks

This repository contains small practice tasks that I created while learning **React.js**.  
The purpose of this project is to understand React concepts by building and practicing step by step.

## What I Learned
- React basics and project setup with Vite
- Components and props
- Import and export
- State management and hooks
- Styling with CSS and Tailwind CSS
- Form handling and two-way data binding
- API calls in React
- useEffect hook basics
- Using LocalStorage
- Basic routing with React Router
- Introduction to Context API

## Purpose
- To practice React fundamentals
- To improve understanding through small tasks
- To build a strong base before moving to advanced projects

## Tech Stack
- React.js
- JavaScript (ES6+)
- HTML
- CSS / Tailwind CSS
